## Основные модели предоставления облачных услуг

- **IaaS (Infrastructure as a Service)** - инфраструктура как услуга. Пользователь получает доступ к виртуальным машинам, и хранилещам.  
- **PaaS (Platform as a Service)** - платформа как услуга. Пользователь аолучает готовую среду для разработки и развёртывания приложений.  
- **SaaS (Software as a Service)** - программное обеспечение как услуга. Готовое приложение по подписке.

## Задача работы

На основе предоставленного набора данных биллинга AWS (файл CSV) надо было классифицировать каждый тип потребления ресурсов по следующим признакам:

- Уровень абстракции (IaaS/PaaS/SaaS)  
- Функциональная категория (IT Tower, Service Family)  
- Детальный тип потребления (Service Type, Service Sub Type, Service Usage Type)

## Ход работы

Для каждой строки таблицы выполнялись следующие действия:
```
1. Определялся сервис по значению в колонке **Product Code** (например, AmazonS3, AmazonMSK и т.д.).  
2. Анализировалась графа **Usage Type**.
3. Значения для колонок **IT Tower** и **Service Family** выбирались из общих категорий AWS  
4. Для определения уровня абстракции смотрел на следующие характеристики сервиса:  
   - Управляемые сервисы без необходимости администрирования серверов - PaaS.  
   - Хранилища и объектные сервисы -  IaaS или тоже PaaS.  
   - Машинное обучение - признак PaaS.  
5. Если на этом этапе было всё ещё не понятно - смотрел на аналогичные Usage Type в примерах.
```

Все строки были обработаны вручную.
